### Install NeoVim
`https://github.com/neovim/neovim/wiki/Installing-Neovim`

### Download the config
- Linux
`git clone https://github.com/shadowmax31/nvim_config ~/.config/nvim`

### Install Vim-Plug
`https://github.com/junegunn/vim-plug`

### Init NeoVim
- Start NeoVim
`nvim`
- Install the plugins
`:PlugInstall`
